# Data-Analysis-Dashboard
## Project Objective
To analyze company employee data to uncover insights related to workforce distribution, demographics, compensation, performance, and organizational growth. This project answers key HR and business questions using data analysis and visualization techniques to support data-driven decision-making.

## Dataset used
- <a href="https://github.com/vkrish099/Data-Analysis-Dashboard/blob/main/hr-data%20(1).xlsx">Dataset</a>

## Questions (KPIs)
- How many people are there in each department?
- Gender distribution by department
- Age spread of our staff (histogram) 
- Min / max / average salary in each department 
- op earners in each country 
- Performance Spread (sort by column) 
- ompany growth trend 
- Employee Filter (by starting letter)
- Performance vs. Salary - is there any relationship? 
- India vs. New Zealand - quick scorecard

- Dashboard Interaction <a href="https://github.com/vkrish099/Data-Analysis-Dashboard/blob/main/hr-data%20(1).xlsx">View Dashboard</a>

## Process
- Verify data for any missing values and anomalies, and sort out the same.
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created pivot tables according to the questions asked.
- Merge all pivot tables into one dashboard and apply slicer to make dynamic.
